Pusat Bantuan & FAQ

Anda perlu mengonfigurasi URL Google Apps Script yang terhubung ke Google Sheets di menu Setelan. Setelah disimpan, data target dan riwayat tabungan akan otomatis tersinkronisasi ke Google Drive Anda setiap kali Anda menambah data baru.
Cukup kosongkan kolom URL Google Apps Script di menu Setelan lalu simpan. Aplikasi akan otomatis beralih ke Mode Offline Lokal dan menyimpan seluruh data Anda secara mandiri di penyimpanan lokal perangkat.
Hal ini sering terjadi jika aplikasi berada dalam Mode Cloud tetapi akun Anda didaftarkan secara lokal (offline), atau sebaliknya. Solusinya, Anda dapat mengosongkan URL Apps Script di setelan layar login untuk beralih ke akun lokal Anda.
Sangat aman. Seluruh data keuangan disimpan langsung di browser perangkat lokal Anda (Mode Offline) atau di Google Sheets akun Google Drive Anda pribadi (Mode Cloud). Kami tidak mengirimkan data Anda ke server pihak ketiga manapun.
Kami sangat menyarankan untuk menggunakan sinkronisasi cloud Google Sheets sebagai metode pencadangan. Dengan begitu, data tabungan Anda tidak akan terhapus meskipun cache browser dibersihkan atau aplikasi dipasang ulang.
